Automated Scheduling System

This Schedule Automation System automatically assigns employees to shifts based on their availability, preferred shift types, and work locations. It reads and manages employee data from a CSV file, ensuring consistency and sorting by hire date for fairness. The built-in scheduling algorithm enforces staffing limits per shift/location while respecting preferences as much as possible. A user-friendly interface also allows updating employee details, and the system can send notifications (e.g., via Twilio) to inform employees of their schedules.

AI Chat Moderator

This Streamlit-based app is a chatroom moderation tool that uses machine learning to detect toxic or harmful content in user messages. It loads pre-trained models and a TF-IDF vectorizer to classify incoming text, flagging messages with high toxicity confidence. The interface allows users to send and view messages in real-time, with flagged messages visually marked and styled for easy identification. It is intended for educational and cybersecurity demonstration purposes, not as a production censorship tool.
